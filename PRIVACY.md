# Privacy Policy

## Tab Fusion

Tab Fusion processes tab and window information locally inside the user's browser in order to:

- merge tabs from multiple windows into one window,
- create manual and automatic tab groups,
- save and restore local snapshots,
- detect and close duplicate tabs.

## Data Handling

- Tab Fusion does not send browsing data, tab URLs, titles, groups, or snapshots to any external server.
- All snapshots and extension state are stored locally using `chrome.storage.local`.
- Tab Fusion does not use analytics, tracking pixels, remote logging, or third-party data processors.

## Permissions

Tab Fusion requests only the Chrome permissions required to perform its local features:

- `tabs`
- `tabGroups`
- `storage`
- `alarms`

## User Control

- Users can delete saved snapshots from the dashboard at any time.
- Users can remove the extension to delete its local behavior and stop future data processing.
- Users can clear extension storage through Chrome extension settings if needed.
